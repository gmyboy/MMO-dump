# PokeMMO Mods
This project is a collection of strings and themes resource for creating Mods for PokeMMO.

# Structure
- `strings/` original strings dump from PokeMMO
- `themes/` original themes dump from PokeMMO

- `strings-zh/` string of simple chinese

- `themes-dark`
